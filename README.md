# Docusaurus-Template

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

## Clone

```sh
git clone https://github.com/hlop3z/docusaurus-template.git && rm -rf docusaurus-template/.git
```

## Install

```sh
npm install
```

## Dev

```sh
npm run dev
```

## Build

```sh
npm run build
```

## Deploy

```sh
npm run deploy
```
