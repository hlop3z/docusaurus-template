# Coding

## Javascript/TypeScript (`tsx`)

```tsx
function main() {
  console.log("Hello, World!");
}
```

## Python (`py`)

```py
def main():
    print("Hello, World!")
```

## Rust (`rust`)

```rust
fn main() {
    println!("Hello, World!");
}
```

## Go (`go`)

```go
package main
import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
